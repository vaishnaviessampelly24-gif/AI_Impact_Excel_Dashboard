\# 🤖 AI Impact on Jobs 2030 Dashboard



\## 📌 Project Overview



This project is an interactive Excel Dashboard developed to analyze the impact of Artificial Intelligence on jobs, workforce trends, salaries, automation, and future job demand by 2030.



The dashboard provides business insights using Pivot Tables, Pivot Charts, KPI Cards, Slicers, VBA Macros, and interactive navigation buttons.



\---



\# 🎯 Objectives



\* Analyze AI impact on different industries

\* Understand automation levels across sectors

\* Study salary trends and future job demand

\* Identify hiring trends and workforce changes

\* Build an interactive and dynamic Excel dashboard



\---
